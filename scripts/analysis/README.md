# Analysis Scripts

This folder contains scripts for analyzing and improving lead quality.

## Scripts

### improved_lead_qualifier.py
Enhanced lead qualification with improved criteria including:
- More sophisticated business validation
- Better revenue estimation
- Improved industry classification

### pre_qualification_filters.py
Pre-qualification filters that screen leads before expensive API calls:
- Basic business criteria checks
- Geographic filters
- Revenue thresholds

## Usage

Run scripts directly:

```bash
python scripts/analysis/improved_lead_qualifier.py
python scripts/analysis/pre_qualification_filters.py
```

Or import as modules in your code.
