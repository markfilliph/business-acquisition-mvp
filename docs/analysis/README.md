# Lead Quality Analysis

This folder contains the latest analysis files for improving lead qualification and quality.

## Files

### Documentation

1. **EXECUTIVE_ACTION_PLAN.md** - High-level action plan for lead quality improvements
2. **lead_quality_analysis_solutions.md** - Detailed analysis of lead quality issues and solutions
3. **100_LEADS_STANDARDIZED_README.md** - Documentation for the standardized 100 leads output
4. **POSTAL_CODES_SUMMARY.md** - Summary of postal code additions to the dataset

### Scripts

Located in `/scripts/analysis/`:

1. **improved_lead_qualifier.py** - Enhanced lead qualification logic with improved criteria
2. **pre_qualification_filters.py** - Pre-qualification filters to screen leads before full validation

## Purpose

These files represent the latest iteration of lead quality analysis and improvement efforts, focusing on:

- Better qualification criteria
- Pre-filtering to reduce API costs
- Standardized output formats
- Postal code enrichment

## Usage

See individual files for specific usage instructions. The scripts are standalone and can be run directly from the command line.
